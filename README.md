**普通用户功能：**

2. 注册
3. 笔记分享
4. 交流互动
5. 公告信息
6. 个人中心
7. 个人笔记管理
8. 笔记分享管理


**管理员功能：**
1. 登录
2.个人中心
3.用户管理
4.笔记分享管理
5.个人笔记管理
6.交流互动

8. 系统管理
